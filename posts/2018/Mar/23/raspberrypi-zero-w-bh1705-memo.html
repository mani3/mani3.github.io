  • raspberrypi

前準備

$ sudo apt-get -y update
$ sudo apt-get -y upgrade
$ sudo apt-get install -y python-smbus i2c-tools

照度センサー

  • BH1750 datasheet

I2C を有効にする

$ sudo raspi-config

// 設定できたら再起動
$ sudo reboot

/boot/config.txt の 差分

$ diff /boot/config.txt /boot/config.txt.origin
46c46
< dtparam=i2c_arm=on
---
> #dtparam=i2c_arm=on

ここにある通り配線してから、 i2cdetect を叩いてデバイスのアドレス 0x23 を確認する

$ i2cdetect -y 1
